Static Mode

What is Static Mode?

Static mode is a feature-limited version of the site I made that does not require a database to run.

Why is the site in static mode?

1. I have to pay for the servers
2. There have never been any donations to help pay for the servers
3. It's cheaper to run the site without a database, so I lose less money every month

What is different about static mode?

Nobody can rate, tag or edit comments. The contact form doesn't work.
Essentially comments will not be updated while the site is like this.

When will the site go back into non-static mode?

When there are more monthly users or there are some donations.
