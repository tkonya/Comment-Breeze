Static Mode

What is Static Mode?

Static mode is a feature-limited version of the site I made that does not require a database to run.

Why is the site in static mode?

This site doesn't make any money and it's cheaper to run without a database.

What is different about static mode?

Nobody can rate, tag or edit comments.
Essentially comments will not be updated while the site is like this.

When will the site go back into non-static mode?

When there are enough users or donations to warrant it.
