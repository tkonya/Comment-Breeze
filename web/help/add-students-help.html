Adding Students:

Enter your student's names, as you would like them to appear in the comments, one after another by selecting their gender, typing their name on the "New Student Name" line and then hitting enter or press the "Add" button after each one.

Tip: You may put a space after the students name and an "m" or "f" to set their gender. For example I could enter in "Jimmy f", hit enter, and the page would add a female student named "Jimmy". If you don't add one of those letters after the student's name to determine gender it will be whatever gender is currently selected in the "Add Students" box.

By default a totally random comment whenever a new student is added, but you may also choose to start blank or generate a comment based on the default pattern (if you have defined a default pattern).

Setting Gender:

You can set the gender of the student to be added one of 2 ways:
1. With the Male/Female/Neutral radio button
2. You may put a space after the students name and an "m" or "f" to set their gender.

Example: For example I could enter in "Jimmy f", hit enter, and the page would add a female student named "Jimmy". If you don't add one of those letters after the student's name to determine gender it will be whatever gender is currently selected in the "Add Students" box.

If you set the gender with a capital letter ("M" or "F") it will also change the default gender. This could be useful if you have several students in a row that are the same gender.

Initial Comment:

By default a totally random comment whenever a new student is added, but you may also choose to start blank.

A third option is to make a default pattern and have all new students added start with comments conforming to the default pattern. Each student added with the default pattern will have a new comment generated based on the pattern.
