Class Name:

The class name is strongly recommeded, as many comments have the variable "CLASS_NAME" in them that should be replaced with something. If your class does not have a name you should start with something generic like "English Class" and then edit it out later if it sounds weird.

Introduction and Conclusion:

The introduction and conclusion are optional sentences that will be added to the beginning and end of each student's comments. They will not initially be visible when writing comments, but when you view finished comments or copy comments they will be appended.

You can use the variables "STUDENT_NAME" and "CLASS_NAME" (no quotes) in them, and they will be replaced with the correct value when copying or viewing finished comments.

Save and Load:

All of your progress can be saved in the Class Details menu. The save file includes your class name, introduction, conclusion, all of your students and their comments, any patterns and settings for each tab. The save file is a normal txt file so your comments can be recovered even if you cannot access Comment Breeze.

You should always save your file after you are done. It will allow you to come back and edit easily if necessary, and it will be another copy of your work.

Note: Windows notepad may not open the save file in an easily human-readable way. Try opening it with basically any other document editor such as Libre Office, Open Office, MS Word, Notepad++, Wordpad, etc...

Loading your saved file will restore all of the things it saved. A good paradigm to follow would be to make a save file for every class you have. This way the next time report cards are due you can just reopen that file and all of your students will be there. You can just regenerate the comments and edit as necessary.

Reset App:

You can reset the application in the Global Details menu. This can be helpful if you already loaded the page but you now have low or no network connectivity. This will reset all fields and erase all students, but will not need to reload the comments from the server. If you are on a shared computer you should clear the cache when done; Reset App has an option to also clear the cache.
