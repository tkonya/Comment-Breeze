Comment Breeze

Welcome to Comment Breeze!

I made this site to help my girlfriend, friends and other ESL teachers build report card comments.
If you're an ESL teacher, I hope this makes your life easier!

What's different about Comment Breeze?

unique comments*
Blazing fast multi-student comment generation
Powerful live search capability

OK, how do I use it?

Go to the Single Student, Multi Student or Smart Search tab.
Click the help button in the upper right hand corner for instructions and tips.
Reading the help for each page first is strongly suggested.

Mobile Warning

I've done my best to make this site responsive and accessible on mobile devices as well as computers.
That said, the page is very javascript heavy and may suffer from slow performance on mobile devices with limited computing power.
Some settings are changed to make mobile viewing experiences better. These can be changed on the settings page. The Smart Search tab is the most CPU intensive of all. I would not even try to use this on mobile; you're phone might ignite.

* All comments are not initially loaded for mobile users, see settings page to change.

How is it made?

Angular JS 1.4 w/ Angular Material 1.0
Java 8 backend, Jersey for REST
MySQL 5.6
Hosted on AWS EBS and RDS in Seoul

settings Settings

Visibility

Tooltips

Turn pop up tooltips on or off. Recommended off for mobile or touchscreen-only users; on for users with a mouse or other pointing device.

Tone Buttons

Show or hide the tone buttons and indicators on the comments list. Gives a little extra space to those who are on small screens or do not need to see or edit tone.

Edit Buttons

Show or hide the edit button on the comments list. Gives a little extra space to those who are on small screens or are not going to edit the comment text.

Tags

Show or hide the tags button and indicator on the comments list. Gives a little extra space to those who are on small screens or are not going to look at or edit tags.


Make Something Up

Adjust the number of comments generated by the "Just Make Something Up" button on the Single Student tab or by default on the Multi Student tab. Can be any number from 1 - 100.


Gender Selection and Switching

Avoid "her"

When auto-generating comments for non-female genders the system cannot know for sure which pronoun to replace "her" with, therefore often times it gets it wrong. Checking this will attempt to avoid comments with "her" when auto-generating comments if the gender is not female. This will reduce the number of available comments for non-females, but also reduce the number of corrections necessary.

Enable Neutral

The gender neutral pronouns are also plural (they, them, etc...) and therefor switching to them from the singular male or female pronouns will often make the sentence grammatically incorrect because verbs will remain singular. You can turn on the ability to use gendered pronouns but more proofreading and correction will be required.


Resize Comments

If searching is slow, generally for mobile users with limited computing power, you can load the page with only some of the comments to improve performance. This is done by default for users on mobile devices. You may reload with any number of comments from 1 to .
