Comment Breeze

Welcome to Comment Breeze!

I made this site to help my girlfriend, friends and other ESL teachers build report card comments. If you're an ESL teacher I hope this makes your life easier.

Why make this?

There are other tools out there, but I thought I could do something different than what I've seen. One thing I've never seen on any other report card site is any kind of search functionality. I wanted to make it so you could search a giant bank of comments for words or phrases that you have in mind. I've never seen a site that had more than a couple hundred comments, so if you were to use that site repeatedly then there wouldn't be much variation in your comments.

What's different about Comment Breeze?

There are currently comments in the database. This is the greatest strength and weakness of this site. The benefit is that there are tens of thousands more comments here than any other site I've seen. Since all of the comments are loaded into your browser upfront, it means that searching them is instantaneous. The downside is that since I don't have the time (or interest) in going through and editing and categorizing all of the comments manually they are not organized or formatted as well as other sites. I have written code to help me go through and format comments in a semi-automated way, but it's not perfect.

OK, how do I use it?

Go to either the Single Student or Multi Student tab. There is a help button in the upper right hand corner that will explain how to use each page. I suggest reading these even if the interface looks intuitive enough at first because there are a couple tricks you should know about.

Mobile Warning

I've done my best to make this site responsive and accessible on mobile devices as well as computers. But if you are visiting this on a mobile site you might want to note a couple things. Firstly the page is very javascript heavy, and may suffer from slow performance on mobile devices with limited computing power. The tooltips are turned off for mobile users due to not having a mouse to hover over things with and also them taking up a much higher percentage of the screen. I currently detect mobile users by looking at whether the screen is less than 1200px wide; if you're seeing this message it means you initially loaded the page under 1200px. You can turn on tooltips in the settings tab.

Visibility

Show Tooltips

Turn pop up tooltips on or off. Recommended off for mobile users; on for users with a mouse or other pointing device.

Show Tone Buttons

Show or hide the tone buttons and indicators on the comments list. Gives a little extra space to those who are on small screens or are not going to edit comment tone.

Show Edit Buttons

Show or hide the edit button on the comments list. Gives a little extra space to those who are on small screens or are not going to edit the comment text.


Make Something Up

Adjust the number of comments generated by the "Just Make Something Up" button on the Single Student tab or by default on the Multi Student tab. Can be any number from 1 - 100.


Resize Comments

If searching is slow, generally on a mobile device, you can reduce the size of the comments you are searching through to increase speed. Select the number of comments you want to search (minimum 1; maximum ) and hit reduce. You can change or restore this at any time. Current size:
