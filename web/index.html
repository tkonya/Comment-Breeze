Comment Breeze

Welcome to Comment Breeze!

I made this site to help my girlfriend, friends and other ESL teachers build report card comments.
If you're an ESL teacher, I hope this makes your life easier!

What's different about Comment Breeze?

unique comments*
Powerful live search
Blazing fast multi-student comment building
Customizable pattern-based generation
Designed for both smartphones and desktops

Mobile Users

Comment Breeze is javascript heavy and may suffer from slow performance on mobile devices with limited computing power.

Some settings are changed to make mobile experiences quicker. These can be adjusted on the settings page.

* All comments are not initially loaded for mobile users, see the "Resize Comments" section of search settings to change.

OK, how do I use it?

Start on the Build tab, there is a help button at the upper right with detailed instructions and tips.
(this may be moved inside the drop down menu on smaller screens)

Patterns allows you to build a sentence-by-sentence template that your comment or comments will conform to. This could be a default pattern applied to all students, or an individual pattern for just one student. Each pattern piece you add will look for one sentence matching the search criteria you set.

Add pattern pieces one by one, each piece represents 1 sentence, in the order you add them.

Example: If you want your first sentence to be about "behavior" then type that in and hit "Add Pattern Piece"

If you don't care what type of comment goes in a certain position just leave the Search Text empty and the filter set to "Any"

Repeat until you have added and configured as many pieces as you want sentences.
