Comment Breeze

Welcome to Comment Breeze!

I made this site to help my girlfriend, friends and other ESL teachers build report card comments.
If you're an ESL teacher, I hope this makes your life easier!

What's different about Comment Breeze?

unique comments*
Blazing fast multi-student comment generation
Powerful live search capability
Servers in Seoul

OK, how do I use it?

There are help buttons on each page at the top or upper right. These have detailed instructions about how each tab works.

Mobile Warning

I've done my best to make this site responsive and accessible on mobile devices as well as computers.
That said, the page is very javascript heavy and may suffer from slow performance on mobile devices with limited computing power.
Some settings are changed to make mobile viewing experiences better. These can be changed on the settings page. The Smart Search tab is the most CPU intensive of all. I would not even try to use this on mobile; you're phone might ignite.

* All comments are not initially loaded for mobile users, see settings page to change.

What happened to the old site?

The old site was too complicated because I had built 1 feature at a time with no cohesive vision of what the end product would be. If you really want to use it still you can find it here: Comment Breeze Classic

Patterns allows you to build a sentence-by-sentence template that your comment or comments will conform to. This could be a default pattern applied to all students, or an individual pattern for just one student. Each pattern piece you add will look for one sentence matching the search criteria you set.

Add pattern pieces one by one, each piece represents 1 sentence, in the order you add them.

Example: If you want your first sentence to be about "behavior" then type that in and hit "Add Pattern Piece"

If you don't care what type of comment goes in a certain position just leave the Search Text empty and the filter set to "Any"

Repeat until you have added and configured as many pieces as you want sentences.
