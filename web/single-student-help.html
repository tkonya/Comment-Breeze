Single Student Comment Building

Setup:

Enter the name of your class and the name of the student you are going to first build a comment for on the "Class Name" and "Student Name" lines respectively. Enter these as you would like them to appear in the comments, so generally just their first/given name. Select what gender the student is.

If you want all comments to start with the same introduction and/or conclusion then enter those on the "Your custom introduction" and "Your custom conclusion" lines. You may use the variables CLASS_NAME and STUDENT_NAME in your introduction and conclusion, they will be replaced with the appropriate values when copied.

Finding a Comment:

Search for a comment you would like to add by typing a word or short phrase on the "Search Comments" line above the list of comments. Click the comment text in the list to add it to your comments. If you are looking for a comment with a certain tone you can select that in the box to the right of the search. Searching may be slow for mobile users or users with limited processing power (you have a crappy computer). Setting comments per page at the minimum value can help slightly.

Just Make Something Up:

The "Just Make Something Up" button will pick a random set of comments from the entire list and set that as your comment (overwriting whatever is currently there). You may select the number of sentences you would like this button to pick on the settings tab.

Editing Your Comment:

Edit comment manually as needed or use the shuffle button to scramble the order of the sentences in it.

Please remember that the gendered pronoun fixing is not perfect due to the male equivalent of "her" being either "him" or "his" and the site just guessing which one it is supposed to be. Proofreading is strongly advised.

Rating and Editing Comments:

Anybody can set the tone of a comment in the list by clicking the icon to the left of it. This can help you and other users organize this list better. If you have the editing password (basically if I know you in real life) then you can click the icon to the right to edit the text. If you don't have the editing password you can still flag a comment for attention if there is something wrong with it.

Copying:

When you are satisfied with the comment you can hit the "Copy Comments" button to copy the entire comment.
The introduction and conclusion will be added to the beginning and end if you entered them. If you put the variables STUDENT_NAME or CLASS_NAME in the intro or conclusion they will be replaced in the copied text.

Subsequent Students:

Hitting the "Reset Comments" button will prepare you to do the next student by erasing the current student name and comments. The class name, gender, introduction and conclusion will not be changed.
