Smart Search Comment Building

Setup:

Enter the name of your class and the name of the student you are going to first build a comment for on the "Class Name" and "Student Name" lines respectively. Enter these as you would like them to appear in the comments, so generally just their first/given name. Select what gender the student is.

If you want all comments to start with the same introduction and/or conclusion then enter those on the "Your custom introduction" and "Your custom conclusion" lines. You may use the variables CLASS_NAME and STUDENT_NAME in your introduction and conclusion, they will be replaced with the appropriate values when copied.

Building a search pattern:

You can build the search pattern sentence by sentence. So for your first sentence enter text you would like to search for, if any, on the "Search Text" line. Check whether you would like to look for this text in the comment text or in the tags. Selecting both will return a comment if it finds the text in either one. Select if you would like this comment to have a certain tone (positive, neutral or negative). Hit "Add Smart Search" and repeat for all other sentences you would like to add. If a search did not return any results or you are unsatisfied with the result, you can modify it below and regenerate.

Using with Multi Student:

A more powerful way to use this feature is to define a search pattern here and utilize it on the Multi Student tab. To do this just make a pattern here, go to the Multi Student tab, make sure "Use Smart Search" is checked, and start adding students as you otherwise normally would there. Instead of generating totally random comments it will generate a comment based on the pattern you've just defined. You can change the pattern at any time as necessary. Multi Students already added will not be affected by pattern changes, but using the regenerate button on any student will regenerate according to the current pattern, assuming "Use Smart Search" is still checked.

Just Make Something Up:

The "Just Make Something Up" button will generate multiple searches using the values you have currently set in the search box.. So if you would like a random set of all negative comments, just select "Negative" in the "Filter on Tone" box and hit "Just Make Something Up" You may select the number of searches you would like this button to pick on the settings tab.

Editing Your Comment:

Edit comment manually as needed or use the shuffle button to scramble the order of the searches that produced it.

Copying:

When you are satisfied with the comment you can hit the "Copy Comments" button to copy the entire comment.
The introduction and conclusion will be added to the beginning and end if you entered them. If you put the variables STUDENT_NAME or CLASS_NAME in the intro or conclusion they will be replaced in the copied text.

Subsequent Students:

Hitting the "Reset Comments" button will prepare you to do the next student by erasing the current student name and comments. This will not erase your searches so you can reuse the values in them for the next student. The class name, gender, introduction and conclusion will not be changed.

Mobile Warning:

This is definitely the most CPU intensive tab in the tool. I would not even try to do this on a smartphone or tablet.

Please Also Note:

There are currently very few comments with tone set or with tags set, so it's not advised to rely on searching/filtering on either of those too much. This tool will become more useful as the comment bank becomes better organized. You can organize it right now by going to the Single Student tab and tagging or rating some comments. Searching on text still works fine though.
